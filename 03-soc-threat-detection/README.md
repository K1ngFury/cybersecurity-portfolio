# SOC Threat Detection and Investigation

## Objective
Simulate common cyber attacks in the homelab and investigate them using the SIEM.

## Lab Setup
Attacks were launched from a Kali Linux machine against systems in the lab network.

## Tools Used
Elastic Stack (SIEM)
Kali Linux
Windows 11
Ubuntu Server

## Attack Simulations
- Brute force login attempts
- Network port scanning
- Suspicious login activity

## Investigation Process
1. Alert generated in the SIEM
2. Review related logs
3. Identify source IP and activity
4. Determine if activity is malicious
5. Document findings

## Skills Demonstrated
- Alert triage
- Log analysis
- Threat investigation
- Incident documentation

## Outcome
Demonstrated the ability to detect and investigate suspicious activity using SIEM logs.
