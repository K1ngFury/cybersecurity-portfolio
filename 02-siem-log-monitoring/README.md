# SIEM Log Monitoring

## Objective
Deploy a SIEM to collect, analyse, and visualise logs from machines inside the homelab network.

## SIEM Platform
Elastic Stack (Elasticsearch, Logstash, Kibana)

## Data Sources
- Windows 11 workstation logs
- Ubuntu server logs
- Network traffic from lab environment

## Configuration
- Installed Elastic Stack
- Configured log ingestion
- Built dashboards in Kibana
- Created basic alert rules

## Skills Demonstrated
- SIEM deployment
- Log ingestion and parsing
- Dashboard creation
- Security event monitoring

## Outcome
Centralised log monitoring environment capable of detecting suspicious activity inside the homelab.
